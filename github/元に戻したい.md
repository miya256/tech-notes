## ステージを外すだけ
ファイルはそのまま
```
git reset HEAD <file_path>
```
## 未ステージのやつを元に戻す
```
git restore <file_path>
```
## 最新コミットの状態に戻す
add したやつも全部消える。
```
git reset HEAD --hard
```
## 直前のコミットを取り消す
コミットだけ消す。addされた状態は維持。
```
git reset --soft HEAD~1
```
## 直前のコミットを取り消す
addしたやつも全部消えます。1つ前のコミットの状態になる
```
git reset --hard HEAD~1
```
