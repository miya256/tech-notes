## 概要
必要なライブラリとかを簡単にインストールするためのもの

## \[project]
### name
プロジェクト名
```
name = "project_name"
```
`pip install name` するときに使われる。
### version
プロジェクトのバージョン。はじめは0.1.0。メジャー.マイナー.パッチ
```
version = "0.1.0"
```
### description
プロジェクトの説明
```
description = "explanation"
```
### readme
```
readme = "README.md"
```
### requires-python
プロジェクトに必要なPythonバージョン
```
requires-python = ">=3.10"
```
### dependencies
必要なライブラリ
```
dependencies = []
```
