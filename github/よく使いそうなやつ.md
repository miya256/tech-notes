## ブランチ一覧
```
git branch // *がついてるのが今いるブランチ
```
## ブランチ作成
```
git checkout -b <branch_name> // 同時に移動もする
```
## commit & push
```
// commitするファイルを追加（file_pathを.にすると変更したすべてのファイル）
git add <file_path>

// commitする
git commit -m "<message>"

// リモートに変更があるなら一応取り込んでおく
git pull origin main

// リモートに反映（-uをつけると、次回から git push だけでよい）
git push [-u] origin <branch_name> 
```
