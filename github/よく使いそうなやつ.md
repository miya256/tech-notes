## 確認
```
// ステージングやコミットの状態確認
git status

// コミット履歴
git log

// 最新コミットとの差分表示
git diff
```
## 元に戻す
### addしてない状態で
```
// 最新コミットの状態にリセット
git reset HEAD --hard

// ファイル単位で最新コミットの状態にリセット
git checkout <file_name>
```
### addした状態で
```
// addしたファイルをリセットして最新コミットの状態に
git checkout HEAD <file_name>
```
### commitした状態で
```
// 最新のコミットをなかったことに
git reset HEAD^
```
