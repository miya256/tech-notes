## ブランチ作成
```
// ブランチを確認。*がついてるのが今いるブランチ。
git branch

// ブランチを作成して移動
git checkout -b <branch_name>
```
### ブランチ削除
```
git branch -d <branch_name> // ローカルの
git push origin -d <branch_name> // リモートの
```
## commit & push
```
// commitするファイルを追加（file_pathを.にすると変更したすべてのファイル）
git add <file_path>

// commitする
git commit -m "<message>"

// リモートに変更があるなら一応取り込んでおく
git pull origin main

// リモートに反映（-uをつけると、次回から git push だけでよい）
git push [-u] origin <branch_name> 
```
## 確認
```
// ステージングやコミットの状態確認
git status

// コミット履歴
git log

// 最新コミットとの差分表示
git diff
```
## 元に戻す
### addしてない状態で
```
// 最新コミットの状態にリセット
git reset HEAD --hard

// ファイル単位で最新コミットの状態にリセット
git checkout <file_name>
```
### addした状態で
```
// addしたファイルをリセットして最新コミットの状態に
git checkout HEAD <file_name>
```
### commitした状態で
```
// 最新のコミットをなかったことに
git reset HEAD^
```
